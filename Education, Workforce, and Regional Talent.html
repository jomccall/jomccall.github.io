Technical Notes

In order to create the larger source dataset, we are pulling a number different datasets together. First, the information from MIT. Second, industry composition and wages. Third, occupational composition and wages. Finally, family composition per geography from the US Census.

Living Wage Data: For the State of Tennessee

Industry Data: Quarterly Census of Employment and Wages, and GNRC's code for calling this data through the Bureau of Labor Statistics API: Github repository

Occupational Data: This information is only available at a State level from the Bureau of Labor Statistics so special request has to be made from partners with proprietary analytics derived from the QCEW and Industry Occupation Matrix. JobsEQ Analytics via the Nashville Area Chamber of Commerce

Family Composition Data: Census Bureau

Repository for Composition of larger dataset: Github
